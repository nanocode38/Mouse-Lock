# Mouse Lock
- [English Version 英文版本](.\README.md)
## 超级鼠标锁

#### 1.功能描述
此工具用于WindowsXP以上的Windows系统， 此工具使用Python编写， 使、用了Pynput、keyboard、pyautogui等库， 感谢库的作者！此项目可以进行静默鼠标锁定， 几乎不占用资源可以长期挂在后台， 60秒不动鼠标、键盘后自动将鼠标锁定在视野中央并锁定键盘， 180秒后启动模拟蓝屏， 在鼠标锁定状态点击鼠标中键即可解锁， 在蓝屏状态右键菜单即可解锁， 蓝屏解锁完成后一定要关闭“摸鱼屏保”程序！否则键盘将处于锁定状态！

#### 2.使用方法
从[https://github.com/nanocode38/Mouse-Lock.git](https://github.com/nanocode38/Mouse-Lock.git)下载最新版EXE程序， 双击运行```Mouse Lock.exe```即可！
