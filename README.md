# Mouse Lock
- [Chinese Version 中文版本](./README-CH.md)

#### 1. Description
This tool is for Windows systems running Windows XP or higher. It is written in Python and uses libraries such as Pynput, keyboard, and pyautogui. Thanks to the authors of these libraries! This project allows for silent mouse locking, with minimal resource usage and can be left running in the background. After 60 seconds of no mouse or keyboard activity, it automatically locks the mouse in the center of the screen and locks the keyboard. After 180 seconds, it triggers a simulated blue screen of death. To unlock during mouse lock, click the middle mouse button. During the blue screen state, right-click to unlock. After the blue screen unlock is complete, make sure to close the "fishing screensaver" program! Otherwise, the keyboard will remain locked!

#### 2. Usage
Download the latest version of the EXE program from https://github.com/nanocode38/Mouse-Lock.git and double-click on the file named "Mouse Lock.exe" to run it!
